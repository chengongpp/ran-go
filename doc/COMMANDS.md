# Commands Reference

```text
0x0000 nope _ equivalent to an empty line
0x0001 version v print version info
0x0100 listnode l list nodes on the mesh network
0x0110 addmapping addm add a mapping
0x0111 delmapping delm delete a mapping
0x0112 listmapping lm list mappings
0x0200 shell sh open interactive shell on target node
0x0201 syncfile scp sync file from source to target
```
